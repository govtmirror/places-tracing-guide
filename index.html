National Park Service

About Places

Places is an internal crowd-sourcing system for the National Park Service's "core" geospatial data. We built Places so that all National Park Service employees, both technical and non-technical, can contribute their knowledge to "the map".

Data from Places is fed directly into both Park Tiles and the larger National Park Service structured data initiative, which will be used to drive NPS.gov and a publicly available application programming interface (or API) starting in January of 2016.

The Places database currently contains the following datasets (click each data type for a snapshot GeoJSON, updated from the database every five minutes):

  1. Points of Interest
  2. Buildings
  3. Roads
  4. Trails
  5. Parking Lots

These datasets were initially seeded from existing park, region, and program-level GIS datasets, but now anyone who is connected to the National Park Service network can edit them using the Places Editor (NPS-only). If your organization has more/better data available and you'd like to get it into Places, send us an email or create an issue on GitHub and we will assist with the upload.

Points of Interest

Point feature

A point of interest or "POI" is represented by a point or "node" on the map and has geographic coordinates (latitude and longitude). A POI can be a standalone node or it can be part of a "way" (an ordered list of nodes representing a linear feature or area). Learn more about POIs from OpenStreetMap's wiki.

Places was originally named "Points of Interest" as POIs were the first (and only) feature type supported. Places supports a wide variety of POIs, a full list of which can be found in our data preset wiki. Don't see the tag you're looking for? Open an issue on GitHub.

This guide is intended to serve as a starter kit for contributing to Places, so we are going to review some of the most common/basic tags to add to the map.

Campground

A campground is an area, usually divided into a number of pitches (or "campsites," see below), where people can camp overnight using tents or camper vans or caravans. Learn more from the OSM wiki here.

Tag a campground with a point in the middle of the area as tourism=camp_site. Feel free to add additional information using the following keys (click for link to OSM wiki):

  • name=
  • operator=
  • addr=
  • opening_hours=
  • capacity=
  • fee=

Lodge

A lodge or "alpine hut" is a remote building located in the mountains intended to provide board and lodging. Learn more from the OSM wiki here.

Tag a lodge with a point on top of the building as tourism=alpine_hut and building=yes. Additionally, trace the building footprint and tag as building=yes (see the buildings section to learn more about tracing building footprints). Feel free to add additional information using the following keys (click for link to OSM wiki):

  • name=
  • operator=
  • addr=
  • opening_hours=

Picnic Site

A picnic site is an area that is suitable for outdoors eating and may have a number of facilities to aid a picnic: tables, benches, restrooms, grills, water taps, covered pavilions, etc. Or, it may just be a grassy area popular for picnics. Learn more from the OSM wiki here.

Tag a picnic site with a point in the middle of the area as tourism=picnic_site. Picnic tables are to be tagged separately, see below. Feel free to add additional information using the following keys (click for link to OSM wiki):

  • name=
  • operator=
  • addr=
  • opening_hours=
  • fee=

Picnic Table

A picnic table is a modified table with benches, expressly for the purpose of eating a meal outdoors (picnicking). Learn more from the OSM wiki here.

Tag a picnic table with a point on top of the table as tourism=picnic_table. Feel free to add additional information using the following keys (click for link to OSM wiki):

Restroom

A restroom or "toilet" open to the public. Learn more from the OSM wiki here.

Tag a restroom with a point on top of the toilet amenities as amenity=toilets and building=yes. Additionally, trace the building footprint and tag as building=yes (see the buildings section to learn more about tracing building footprints). Feel free to add additional information using the following keys (click for link to OSM wiki):

  • name=
  • operator=
  • addr=
  • opening_hours=
  • toilets:disposal=
  • gender (male=, female=, or unisex=)
  • indoor=
  • access=

A waste basket is a single small container for depositing garbage that is easily accesible for pedestrians. Similar expressions: rubbish bin, litter bin, trash can, garbage can. Learn more from the OSM wiki here.

Tag a waste basket with a point on top of the container as amenity=waste_basket. Feel free to add additional information using the following keys (click for link to OSM wiki):

  • name=
  • operator=
  • addr=
  • waste=
  • bin=

Trailhead

A trailhead is the point at which a trail begins, where the trail is often intended for hiking, biking, skiing, snowshoeing, snowmobiling, horseback riding, or off-road vehicles. There is currently no OSM wiki for trailhead available.

Tag a trailhead with a point on the location of the start of the trail as highway=trailhead. Feel free to add additional information using the following keys (click for link to OSM wiki):

  • name
  • operator
  • tracktype
  • surface
  • width
  • incline
  • mtb:scale
  • mtb:scale:uphill
  • mtb:scale:imba
  • sac_scale
  • trail_visibility
  • access

Visitor Center

A visitor center is a building housing information for visitors, typically including some or all of the following: information resources, facilities (e.g. restrooms, drinking water), ranger staff, and exhibits. Learn more from the OSM wiki here.

Tag a visitor center with a point on top of the building as tourism=information, information=office, and building=yes. Additionally, trace the building footprint and tag as building=yes (see the buildings section to learn more about tracing building footprints). Feel free to add additional information using the following keys (click for link to OSM wiki):

  • name
  • operator
  • addr=
  • opening_hours=

Information

Information is an information source for visitors, typically an informational board. This is the default and most basic information-related tag, which can be specified further by including other tags for visitor centers (information=office, building=yes) or information maps (information=map). Learn more from the OSM wiki here.

Tag information with a point on top of the amenity as tourism=information. Feel free to add additional information using the following keys (click for link to OSM wiki):

  • name
  • operator
  • addr=
  • opening_hours=

Park Tiles rendering:

Ranger station

A ranger station is a building or facility primarily intended to house park law enforcement and rescue staff, and is typically not open to the public. It can also be as simple as a ranger sitting at a campground kiosk. Ranger stations can be seasonal, and they are commonly confused with visitor centers as both can be located in the same building. In the event the two are co-located, the best practice is to create two coincident points - one for the ranger station and another for the visitor center. Learn more from the OSM wiki here, but keep in mind that definitions can vary between the Park Service and Forest Service, and between parks.

Tag a ranger station with a point on top of the building as amenity=ranger_station and building=yes. Additionally, trace the building footprint and tag as building=yes (see the buildings section to learn more about tracing building footprints). Feel free to add additional information using the following keys (click for link to OSM wiki):

  • name
  • operator
  • addr=
  • opening_hours=

Telephone

A telephone is a phone available for public use, typically a pay phone on a stand or wall. It may be indoors or outdoors. Learn more from the OSM wiki here.

Tag a telephone with a point on top of the telephone as amenity=telephone. Where several telephones are arranged side by side, arrange several similar points side by side. Feel free to add additional information using the following keys (click for link to OSM wiki):

  • name
  • operator
  • addr=
  • indoor=

Food service

Food service is what we commonly refer to as a food court. It is typically an indoor area within a facility that houses multiple food vendors and provides a common area for dining. Learn more from the OSM wiki here.

Tag food service with a point on top of the building housing the amenity as amenity=food_court and building=yes. Additionally, trace the building footprint and tag as building=yes (see the buildings section to learn more about tracing building footprints). Gas station

A gas station is a facility where vehicles can be refueled. Learn more from the OSM wiki here.

Tag a gas station with a point on top of it as amenity=fuel and, if applicable, building=yes. If the building houses a store, tag that with a point on top of the building as shop=general. Additionally, trace the building footprint and tag as building=yes (see the buildings section to learn more about tracing building footprints).
